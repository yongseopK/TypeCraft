# Developer MBTI

개발자 성향 검사 서비스

## 프로젝트 구조
```
developer-mbti/
├── frontend/    # Next.js 14
├── backend/     # Spring Boot 3
└── docs/        # 기획 문서
```

## 로컬 개발 환경

### Prerequisites
- Node.js 18+
- Java 17+
- MariaDB 10.11+

### 시작하기

**1. 백엔드 실행**
```bash
cd backend
톰캣 실행 ㅇㅇ
```

**2. 프론트엔드 실행**
```bash
cd frontend
npm install
npm run dev
```

**3. 브라우저에서 확인**
- Frontend: http://localhost:3000
- Backend API: http://localhost:8080
